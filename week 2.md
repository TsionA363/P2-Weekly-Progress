# PHP Basics

## Variables & Data Types
- Variables in PHP start with `$` and can hold different data types like strings, numbers, or arrays.

## Constants
- Constants are defined using `define("NAME", value)` and cannot be changed after creation.

## Output Methods
- `echo` is used to print text or variables to the screen. It can concatenate multiple strings using `.`.
- `print` is similar to `echo` but can only take a single argument and returns `1`.

## String Operations
- String concatenation allows you to join variables and text into a single string.

## Arrays
- Arrays store multiple values in a single variable, which can be accessed using indexes.

## Loops
- Loops like `foreach` let you go through each item of an array and perform actions on them.

## Operators

### Arithmetic Operators
- PHP operators include arithmetic (`+`, `-`, `*`, `/`) to perform calculations.

### Comparison Operators
- Comparison operators (`>`, `<`, `==`, `!=`) are used to compare values.

### Logical Operators
- Logical operators (`&&`, `||`, `!`) help combine multiple conditions in `if` statements.
